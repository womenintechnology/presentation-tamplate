Hexadecimal

Hexadecimal RGB value: Six value from 0 - 9 and a - f. Each pair represent a variation of colors: red; green; blue with 256 combination (16 * 16 = 256);

RGB & RGBA

RGB & RGBA: red-green-blue /-alpha: value for each color in notation of 255. “A” value is for transparency from 0 to 1.

Example RGB

Example RGBA

rgba(255, 153, 153, .5)

Responsive Web Design

Elements on the page try to fit to the width of device's screen. It is much more usable (UX) for users.

Two ingredients:

  1. Meta tags in 'head' section in out html;
  2. Media Queries (conditional CSS): it's a way to show our css how to proceed on a user screen of a certain siz;

Meta tags

more info

Responsive meta tags
  • name=”viewport”: It refer to web browser's window area, which is screen of user’s devises.
  • content=”width=device-width, initial-scale=1”: We want the browser window to function at the actual screen width and to be truthful, not pretend to be larger.

These tags tell the device browser that it does not have to conform to the site's CSS rules. The site needs to be responsive and adaptable to the size of the device.

Media Queries


            @media screen and (max-width: 340px) {
              .container {
                  margin: 0 -20rem;
                  padding: 2rem 20.5rem;
              }
           }

Media Queries are conditional css, what gives as opportunity to set style to specific viewport width.

Responsive Design Methodology

  1. Mobile first: start from styling for mobile, and for bigger sizes use media query min-width
  2. Desktop first: start from styling desktop, then smaller screen like mobile or tablet with max-width attributes in media query
